ctrl+alt+t-открытие терминала на ubuntu
/- символ корневой директории
~- символ домашней директории (тильда) 
1.pwd (print working directory)-> выводит путь до текущей директории (папка)
2.ls(list)-> выводит список всех директорий и файлов в текущей рабочей директории
3.cd (change directory)-> меняет текущую рабочую директорию на указанную (cd<path>) 
4.touch-> создает указанный файл (touch <name_of_file>)
5. ls -a -> выводит список скрытых и обычных файлов и директорий
6. mkdir (make directory)-> создает указанную директорию (mkdir <name_of_directory>) 
7. rmdir (remove directory)-> удаляет директорию (rmdir<name_of_directory)
8. rm (remove)-> удаляет указанный файл
9. ls -la -> выводит список всех директорий и файлов с подробной информацией
10. rm -rf -> принудительное удаление директории
15. rm -r -> удаление папки со всем содержимым
11.cat <path_to_file>- >выводит содержимое файла
12.nano <path_to_file> -> открывает графический интерфейс для редактирования файла
*если нет указанного файла, то он создаст новый файл с таким именем
*если мы хотим чтобы в названии был пробел то прописываем через '\' ->mkdir Hello\ World
13.cp (copy)-> копирует содержимое файла на другой (cp <name_of_the_file name_of_the_file>) 
14.mv (move)-> перемещаеи файл в папку (mv <name_of_file.txt ../name_of_folder>)
