1. git init -> создание файла настройки. GIT
2. git add <filename> -> добавление файла на загрузку
2.1 git add. -> добавить все файлы в папке на загрузку
3. git commit -m '<message>' -> подтверждает изменения в файлах, и окончательно готовит их к загрузке
4. git remote add <link name> <ssh link> -> пишется один раз после git INIT, добавляет ссылку на репозиторий в конфигурации
5. git push origin <branch name> -> загрузить изменения, совершенные в git commit, на сайт

Для загрузки новых файлов повторить шаги (2,3,5)
